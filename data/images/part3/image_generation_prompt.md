# Gemini 画像生成プロンプト (金融バナー広告)

1回のプロンプトで8枚のバナーを4列×2行グリッドで1枚の画像に生成する。
生成後、Python/Pillow で個別画像に切り出す。

分析カラム: has_person / has_mascot / color_tone / is_complex / has_number_highlight

---
