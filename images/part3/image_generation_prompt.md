# Gemini 画像生成プロンプト (金融バナー広告)

1回のプロンプトで8枚のバナーを4列×2行グリッドで1枚の画像に生成する。
生成後、Python/Pillow で個別画像に切り出す。

分析カラム: has_person / has_mascot / color_tone / is_complex / has_number_highlight

---

## Batch 1 (ad_001〜008)

```
以下8枚の金融商品Webバナー画像を4列×2行のグリッドレイアウトで1枚の画像にしてください。
各バナーは正方形（1:1比率）。実際の日本の金融会社が使うようなプロフェッショナルなデザイン。
フォントやコピーは日本語で明確に読めること。ロゴ・社名は架空で可。
人物は全員日本人。複数の人物バナーが含まれるが、それぞれ外見・年齢・性別が異なる別人にすること。

上段左：【投資信託】ダークブルー基調。ペンギンのキャラクターがグラフの前に立っているマスコットデザイン。人物なし。「長期運用で資産を育てる」などのキャッチコピー。数字の強調なし。シンプルなレイアウト。
上段中左：【NISA】ネイビー×ホワイト基調。棒グラフと運用実績の数値表が並ぶ。人物なし、マスコットなし。「年率◯%」「累積リターン+◯%」など数字を複数大きく強調。情報量多め。
上段中右：【生命保険】明るいオレンジ×ホワイト基調。笑顔の30代日本人夫婦（夫・妻）と小学生の子供1人が写る。家族3人が画像の50%を占める。マスコットなし。グラフなし、数字の強調なし。安心感を訴求するコピーのみ。
上段右：【がん保険】明るいオレンジ×ホワイト基調。かわいいクマのマスコットキャラクターが中央に配置。人物なし。「月々◯◯円から」の数字を大きく強調。グラフなし。シンプル。

下段左：【クレジットカード】ダークネイビー×ゴールド基調。カードの実物画像が中央に大きく配置。人物なし、マスコットなし。「年会費永年無料」「ポイント還元率◯%」の数字を強調。シンプル。
下段中左：【クレジットカード】レッド×ホワイト基調。元気なキツネのマスコットキャラクターがカードを手に持って笑っているデザイン。人物なし。「年会費無料」「最大◯%還元」の数字を強調。シンプル。
下段中右：【医療保険】グリーン×ホワイト基調。安心した表情の50代日本人女性（ショートヘア・落ち着いた服装）が画像の40%を占める。マスコットなし。グラフなし、数字の強調なし。シンプル。
下段右：【iDeCo】ネイビー×ゴールド基調。折れ線グラフと円グラフが並ぶ複雑なレイアウト。人物なし、マスコットなし。「節税◯万円」「想定利回り◯%」など数字を複数強調。情報量多め。
```
